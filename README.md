# 📦 Listagem de Encomendas

Sistema de gestão e impressão de encomendas com atualização automática.

### 🔹 O que faz:
* **Dados:** Liga-se por **ODBC** e usa **SPs** para processar listagens e impressões.
* **Impressão:** Visualização profissional estilo **Crystal Reports**.
* **Launcher:** Verifica versões e atualiza a app automaticamente via **GitHub Releases**.

### 🛠️ Como preparar:
1.  Compilar o **Launcher**.
2.  Compilar a **App Principal**.
3.  Zipar a App e lançar como **Release** no GitHub.
